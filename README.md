# 🎮 Tic Tac Toe Game

A simple, interactive, and responsive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. The game allows **two players to enter their names** before starting and enjoy a classic Tic Tac Toe match with automatic winner and draw detection.

## ✨ Features

- 👥 Two-player gameplay
- 📝 Player name input before the game starts
- ❌⭕ Displays the current player's turn
- 🏆 Automatically detects the winner
- 🤝 Detects draw situations
- 🔄 Restart/New Game functionality
- 📱 Responsive and user-friendly interface
- 🎨 Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎮 How to Play

1. Enter the names of **Player 1** and **Player 2**.
2. Start the game.
3. Player **X** makes the first move.
4. Players take turns placing **X** and **O** on the board.
5. The first player to get **three marks in a row** (horizontal, vertical, or diagonal) wins.
6. If all nine cells are filled without a winner, the game ends in a **draw**.
7. Click **Restart/New Game** to play again.
