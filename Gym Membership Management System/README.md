# 🏋️ Gym Membership Management System

A modern and responsive **Gym Membership Management System** frontend developed using **HTML, CSS, and JavaScript**. This project is designed to simplify gym management by providing an intuitive interface for managing memberships, trainers, classes, attendance, and payments.

## 📌 Project Overview

The Gym Membership Management System provides an easy-to-use platform for gym members and administrators. It features a clean, responsive design suitable for college projects, demonstrations, and frontend practice.

## ✨ Features

- 🏠 Responsive Landing Page
- 👤 Member Registration & Login
- 📋 Membership Plan Selection
- 👨‍🏫 Trainer Management
- 📅 Class Schedule & Booking
- ✅ Attendance Tracking
- 💳 Payment Interface (Demo)
- 📊 Admin Dashboard
- 📈 Analytics Dashboard
- 💪 Workout Plans
- 🥗 Nutrition Plans
- 🔔 Notification System
- 🌙 Dark Theme UI
- 📱 Fully Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Local Storage

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Gym-Membership-Management-System.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional installation or dependencies are required.
